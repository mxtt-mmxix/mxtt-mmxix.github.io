Variables

Variables - (noun) An element, feature, or factor that is liable to vary or change and in computer code, variables are used for storing data in which its value may be changed.

Variables can hold any data type, so we must know what they are. The data types that all programmers should know are the primitive data types, which as the name implies, these data types are the most simplest.

The primitive data are:

  • int - This data type can only store data values that are negative or positive integers and nothing else.
  • double - This data type can only store data values that are real numbers. (Floats do exist in Java however are depreceated.)
  • boolean - This data type may only have two values: true or false

What happens when you try to store data that does not match the data type of the variable?

When we assign a variable its type, the memory allocated is specialized for that type only, so if you try to assign a value whose type is different, the JVM will through a 'type mismatch' error. Think about it this way, if someone told you to drill a 5 mm hole for a screw, you drill the hole, but they gave you a 10 mm screw. It wont fit.


Initializing, Declaring, and Assigning Variables

To declare a variable, you must follow this format.
Type name = initial value (if any);


Example of initializing variables..

public class nameOfYourFile {
 public static void main(String[] args) {
  int meaningOfLife = 42;
 }
}

Rules when naming a variable:

  • Variable names should only consist of alphanumeric characters. (Some exceptions apply)
  • Variable names must start with a letter.
  • Variable names should be consistent.

Please note, different variables types have different formatting. For example, regular variables should be named similar to variableName, whereas final variables should be named similar to VARIABLE_NAME.
