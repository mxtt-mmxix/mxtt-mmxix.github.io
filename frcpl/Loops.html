Loops

Loops

Loops allow us to repeat code numerous times without having to retype the code every single time.

Their purpose is to reduce redundancy which basically means loops reduce repeated code. To elaborate, whenever programmers type up code, and they need to make the code do a certain function multiple times in a row, it is likely that code will be repeated to make the desired function occur. For example, setting a robot to move forward for a certain amount of time.

While Loops

while (condition)
{
 //Code goes here.
}

A while loop will execute a body of code depending if the condition it checks remains true otherwise it will not execute the body of code it has.

They are used for making specific bodies of code execute multiple times in a row for an undetermined amount of times that you don’t know. For example, imagine there’s a robot has a sensor that detects proximity to how close the robot is to the wall. And your goal for the robot is for it to move to the wall and then stop at the wall. Thus you should write for the condition that the value of a sensor should not be exceeded in which the robot may stop at a certain point near the wall.

For Loops

for (iterable; condition; step)
{
 //Code goes here.
}

Example:

for (int i; i < 16; i++)
{
 //Code goes here.
}

A for loop has 3 parts that must be completed.

Iterable

This part requires that you have a numeric variable declared. Which means you need to use variables that have the int or double variable type. Then type a semicolon after the declaration of the variable. Note that this variable’s value may only be accessed by the body of code in the for loop.

Condition

This part requires you to incorporate the local variable you’ve declared in part one. This will make your loop execute a certain amount of times. Then type a semicolon after the declaration of the variable.

Step

This part requires you to set how the local variable you’ve declared in part one is affected. By setting how the variable is affected, this can affect how many times the loop iterates.


For loops are useful for making specific bodies of code execute multiple times in a row for a set amount of times that you want the loop to occur. For example, you want a robot to move forward 3 inches. And of the given code, the body of code will move the robot 1 inch so you want that code to repeat 3 times as to make the robot move 3 inches.


To prematurely exit a loop, use the "break" keywork.

for (iterable; condition; step)
{
 if (condition) //Execute something...
 else break;
}
