Operators

Operators in Java are modifiers that can be used quickly and easily.

  • The -- operator will take a numerical value and reduce the value by 1.
  • The ++ operator will take a numerical value and increase the value by 1.
  • The > operator checks two values and to see if the value to the left of the > operator is greater than the value to the right of the > operator. If the value on the left is greater than the right value then it will return true, if not false.
  • Likewise, with the > operator, the < operator checks two values and to see if the value to the left of the < operator is less than the value to the right of the < operator. If the value on the left is less than then it will return true, if not false.
  • The = operator is called the assignment operator in which is used for assigning variables, or objects values
  • The == operator is different from the assignment operator as this operator checks between 2 values and determines if they refer to the same object or of variables have the same value. In which if the values are the same, then this operator will return true, if not, false
  • The != operator hecks between 2 values and determines if they don’t refer to the same object or of variables have the same value. In which if the values aren’t the same, then this operator will return true, if not, false
  • The && and || allows for complex conditionals in which checks two conditions to be checked. But there’s a difference between the && and || operators. The && must have the two conditionals true in order to give a final value of true.

    Example
    (true && true) will return true, (true && false) and (false && true) and (false && false) will return false.
    IN A IN B OUT
    true true true
    true false false
    false true false
    false false false

    The || operator must have one of the two conditionals true in order to give a final value of true.

    Example
    (true || true) and (true || false) and (false || true) will return true, (false || false) will return false
    IN A IN B OUT
    true true true
    true false true
    false true true
    false false false

More Operators!

  • The += operator will take the current value of the variable being assigned and add the value that is assigned to it.
  • The -= operator will take the current value of the variable being assigned and subtract the value that is assigned to it.
  • The *= operator will take the current value of the variable being assigned and multiply the value that is assigned to it.
  • The /= operator will take the current value of the variable being assigned and divide the value that is assigned to it.
  • The %= operator will take the current value of the variable being assigned and modulo the value that is assigned to it.
