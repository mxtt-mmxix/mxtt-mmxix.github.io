A Brief Concept of Computer Coding

Before we start learning about how to write code, we’ll need to first understand how coding works and how programmers make use of their code. There are different forms of code. And the lowest is: Machine Code, and this is code in which only the computer understands where we humans do not understand. And then there’s High Level Code that are code that we can understand that take form as programming languages like Java. Which we will be using in the near future.


To explain

Computers and humans have different programming languages we understand to make technology work. Thus we need a way to communicate our code to computer code to make the computer do what we want. And the method to do so is, we have something called a compiler in which our code is transformed into code that the computer understands.


To make an analogy

We programmers who code in java will be like Person A who only speaks English. While the computer that only understand machine code will be like Person B who only Chinese or Spanish. Now in this scenario, Person A needs Person B to do a certain task, but Person A can’t communicate with Person B of the task that Person B needs to do. Thus a translator needs to translate information from Person A to Person B and every time person A needs Person B to do something, a translator is needed. This is the same concept with us programmers compiling our code with each new code as to translate the code we can utilize into machine code that the computer will understand.
