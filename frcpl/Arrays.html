Arrays

Arrays

Arrays are basically a collection of data. It’s more easy to remember arrays as collection of values of the same variable type.

How To Declare Arrays

The initialization of arrays are quite similar to initializing variables, in which there are slight differences.

Rules for declaring arrays are as follows:

  • Writing brackets [] at the end of variable type, and for assigning values to array, the values must be surrounded by curly brackets {}.
  • The values of the array must match the variable type declared. Also, when inputting values into the array, the values must be separated by commas.

Indexes

When there are multiple values of an array, and you want to access a certain value of the array, you may find the index, the place where a specific value may be found. But one key thing you must know is that, the first value of the array starts with an index of zero, and the next consecutive values follow through.

String[] example = {"Hello", "World", "What’s", "Up?", "The sky!"};

The first value, “Hello”, has the index 0.
The second value, “World”, has the index 1.
The third value, “What’s”, has the index 2.
The fourth value, “Up?”, has the index 3.
The fifth value, “The sky!”, has the index 4.
