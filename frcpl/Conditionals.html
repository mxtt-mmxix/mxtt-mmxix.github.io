Conditionals

Conditionals are used for checking boolean expressions, in which conditionals do the function they imply which is to check whether something is true or false.

The syntax for conditionals are simple:

if (condition)
{
 //code goes here
}

If you wish to have more conditions on following the first condition, simply write:

if (condition)
{
 //code goes here
}
else if (condition0)
{
 //more code goes here
}

If no conditions are met, you should write:

if (condition)
{
 //code goes here
}
else if (condition0)
{
 //more code goes here
}
else
{
 //even more code goes here
}

Note:

You must use curly brackets to show where the code starts and where the code ends. If this is not done correctly you will run into buildtime errors.

Another thing to keep in mind is that if the if condition or an else if condition is met, the following else if and else statements will not be executed along with the code of the else if/else statements

if (condition)
{
 //This will run, the code will exit the if statement.
}
else if (condition0)
{
 //Regardless of trueness of this conditional statement. It will not run as the first conditional has already been executed thus this conditional will be ignored.
}

Another thing to keep in mind is that there is something called a nested conditional. They are conditionals inside other conditional.
