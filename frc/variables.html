Variables

Last lesson, we said data stored on disk is are stored in files, but what about data stored in memory? Introducing variables, a means of interacting with data on computer memory. Variables are simply references to data stored in memory. Let's look at some examples.

The declaration of variables is as follows Type name;. The initialization of variables is as follows Type name = value;. On line 3, we wrote int x;. This tells the compiler to allocate 4 bytes of memory, and associate the symbol x with the memory location of those 4 bytes. Then we wrote x = 69;. This tells the compiler to look up the memory address associated with x, and assign the value 69, or 00000000 00000000 00000000 01000101 to the memory address associated at x. Likewise, on line 5 we typed double y = 3.14;. This tells the compiler to allocate 8 bytes of memory, assign its memory location to symbol y, then assign the value 3.14, or 0 00000000000 0000 00000000 00000000 00000000 00000000 00000001 00111010 to that memory address.

Data Types

Java has 8 primitive data types, they are:

Data Type Size Description
byte 1 byte Stores whole numbers from -128 to 127
short 2 bytes Stores whole numbers from -32,768 to 32,767
int 4 bytes Stores whole numbers from -2,147,483,648 to 2,147,483,647
long 8 bytes Stores whole numbers from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
float 4 bytes Stores fractional numbers. Sufficient for storing 6 to 7 decimal digits
double 8 bytes Stores fractional numbers. Sufficient for storing 15 decimal digits
boolean 1 bit Stores true or false values
char 2 bytes Stores a single character/letter or ASCII values

In Java, Each type specifies a specific memory layout. Java, unlike Python, is a statically-typed language, which means types are checked at compile time. Furthermore, unlike Python, Java is also strongly typed. However, these are not the only types in Java. Java has thousands of other non-primitive data types such as String and ArrayList. You can also specifiy your own data types using classes.
