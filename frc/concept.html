Concepts

How and why programming works.

Machine Code

You guys may know that computers operate using 1s and and 0s. This is true as 1s and 0s, when combined can create thigs like 00010101 11011111 or 5599, in decimal. In short each instruction such as ADD, SUBTRACT, and DIVIDE have a number associated with it. Of course, writing 1s and 0s gets tedious over time, so we created hexadecimal. 00010101 11011111 translates to 15DF, much shorter, which is still 5599. By all means the number 5599 is still represented as 00010101 11011111 in computer memory, but we will simply say 15DF, because it is much shorter and easier to work with. Same way we say idk instead of i don't know, because that is easier to work with, and you guys probably have no idea about what I am talking about. Assembly

Introducing Assembly language. This much better than just 1s and 0s (Or hexadecimal if you will). But it still is not quite clear what it does, however feel free to guess what it does. Eventually programmers decided they need an even better language to work with, so they created a new higher level programming language, called C.

It should be noted that assembly is a low-langauage and is therefore not portable. Assembly is nothing more than a human readble form of binary, and therefore is bound to to the specific architecture of the processor. ARM assembly wont run on an x86 processor and vice versa.

High Level Langauges

This is much better, it is way easier to read and write. The C programming language was created by Bell Labs (subsidiary of AT&T) in 1972 to create UNIX, the most powerful operating system even today! That's right, UNIX is so powerful that has withstood the test of time and is still the most powerful operating system even 50 years after its creation. Such a powerful operating system, needed a equally powerful language, thus C was created to create UNIX. We will not be using C however, but Java.

Java was one of the first languages to feature an object-oriented only paradigm.

C and Java are the not the only programming lanuages however. there are plenty of others for you to try out.

FORTRAN

FORTRAN was the first high-level language ever. C was the second. FORTRAN was developed by IBM in 1957; That's right, FORTRAN has been around before the NASA even considered landing on the Moon.
